state value 可以用来评价一个策略的好坏，如果：

$$v_{\pi 1}(s) >= v_{\pi 2}(s)$$

可以说 $\pi_1$ 优于 $\pi_2$

定义：

一个策略 $\pi^*$ 如果对任意的状态有 $v_{\pi^\*} >= v_{\pi}(s)$

需要解决的问题：

+ 最优策略是否存在

+ 最优策略是否唯一


## Bellman optimality equation(BOE)，贝尔曼最优公式

$$v(s)=max_{\pi}\sum_{a}{\pi(a|s)} (\sum_{r}{p(r|s,a)r + r\sum_{s'}{p(s'|s,a)v(s')}})$$
